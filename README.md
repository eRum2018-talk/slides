# eRum 2018 - Slides

To build the slides markdown file (`index.Rmd`), you need to install the
[Jumping Rivers](https://jumpingrivers.com) style package:

```
install.packages("drat")
drat::addRepo("jr-packages")
install.packages("jrPresentation")
```
